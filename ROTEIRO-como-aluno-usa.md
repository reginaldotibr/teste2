❯ como o aluno usa o ROTEIRO.md no laboratório                                                                                                            

● O aluno abre o ROTEIRO.md no VS Code e segue os passos. Tem duas formas de visualizar:
  
  ---
  Opção 1 — Visualizar formatado (recomendado)

  No VS Code, clica com o botão direito no arquivo ROTEIRO.md e escolhe "Open Preview" (ou Ctrl+Shift+V). O arquivo aparece formatado com títulos, blocos
  de código destacados e tabelas — muito mais fácil de ler.

  ---
  Opção 2 — Dividir a tela

  Abre o ROTEIRO.md em preview num lado e o terminal no outro. O aluno lê o passo, copia o código ou comando, e executa — sem precisar alternar janelas.

  ---
  Fluxo prático na aula:

  1. Você distribui o arquivo ROTEIRO.md (pen drive, Google Drive, repositório)
  2. O aluno abre no VS Code em modo preview
  3. Segue passo a passo: cria pasta, roda os comandos no terminal integrado (Ctrl+``  ``), cria cada arquivo com o conteúdo do roteiro
  4. No passo 27 roda node server.js e abre http://localhost:3000 no navegador
  5. No passo 28 cadastra o primeiro usuário via terminal e faz login

  ---
  Dica para a aula: peça para os alunos abrirem dois painéis no VS Code — preview do roteiro à esquerda e o arquivo que estão editando à direita. Assim
  copiam o código sem perder o contexto de onde estão no roteiro.
